SQL Generator

SQL can be a tricky thing. The syntax and keywords change from platfom to platfom and sometimes even version to version. Knowing how to write every kind of SQL query takes hours writing queries and reading documentation. That's where this site comes in. SQLGenerator.net helps you write those tricky SQL statements. Right now we only support the creation of PIVOT statements for SQL Server; but we have plans for more statements and platforms.

I hope you find this site useful. If you have any questions or comments feel free to hit me up over on my website or twitter. Enjoy.
